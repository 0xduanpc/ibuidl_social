# 部署命令

## 编译

anchor build

## 部署

anchor deploy

## 测试

anchor run api

## 查询 spl token 余额

spl-token accounts [mint account]
例如：spl-token accounts 5mxpDqs73g4aMupesL9T5UJwvQ319MT1QQRQVMAUfxn7
